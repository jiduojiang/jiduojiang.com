How to use it

in any new project

In every project's index.html head:

<link rel="stylesheet" href="/shared/fonts.css">
<link rel="stylesheet" href="/shared/tokens.css">
<link rel="stylesheet" href="/shared/components.css">

<!-- then use the classes -->
<button class="btn btn--primary">Start</button>
<div class="card" style="--card-bg: var(--c3)">...</div>

The three rules:

  1. Never hardcode colors. Use var(--c1..--c6).
  2. Pick one crayon per project. Use it for the emoji block, primary button, and any accent.
  3. Compose components. Don't restyle them. To change the look, edit tokens.css.

Full reference lives in shared/DESIGN.md — Claude Code reads it before starting any new project.
